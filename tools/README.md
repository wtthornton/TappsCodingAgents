# Project tools

## Beads (bd)

[Beads](https://github.com/steveyegge/beads) is installed here for dependency-aware task tracking.

- **Run:** `tools\bd\bd.exe` (Windows) or `tools/bd/bd` (if a Unix binary is added). Add `tools/bd` to PATH to use `bd` directly.
- **Init:** Already run with `bd init --stealth` (`.beads/` is local only).
- **Quick start:** `tools\bd\bd.exe quickstart`
