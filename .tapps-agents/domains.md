# TappsCodingAgents Project Domains

This file defines the business domains for the TappsCodingAgents framework project and their primary experts.

## Project: TappsCodingAgents - AI Agent Framework

A comprehensive framework for building and orchestrating AI coding agents with workflow management, industry experts, and code quality tools.

### Domain 1: AI Agent Framework

- **Primary Expert**: expert-ai-frameworks
- **Description**: AI agent orchestration, workflow management, agent communication patterns, multi-agent systems
- **Key Areas**: Agent lifecycle, state management, inter-agent coordination, workflow execution

### Domain 2: Code Quality & Analysis

- **Primary Expert**: expert-code-quality
- **Description**: Static analysis, code scoring, quality metrics, refactoring patterns, code review automation
- **Key Areas**: AST parsing, code metrics, quality thresholds, improvement recommendations

### Domain 3: Software Architecture

- **Primary Expert**: expert-software-architecture
- **Description**: System design patterns, architecture decisions, technology selection, API design
- **Key Areas**: Design patterns, microservices, service boundaries, scalability patterns

### Domain 4: Development Workflow

- **Primary Expert**: expert-devops
- **Description**: CI/CD integration, testing strategies, deployment patterns, development workflows
- **Key Areas**: Test automation, deployment pipelines, environment management, quality gates

### Domain 5: Documentation & Knowledge Management

- **Primary Expert**: expert-documentation
- **Description**: Technical documentation, API documentation, knowledge bases, documentation generation
- **Key Areas**: Doc generation, knowledge base management, documentation standards, RAG integration
