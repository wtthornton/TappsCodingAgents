## Installation

All required dependencies are included in `pyproject.toml`:

```toml
[project.optional-dependencies]
dev = [
    "pytest>=9.0.2",
    "pytest-sugar>=1.1.1",      # Colored output
    "pytest-html>=4.1.1",       # HTML reports
    "pytest-rich>=0.2.0",       # Rich output (optional)
    "rich>=14.1.0",              # Rich library
]
```

Install with:
```bash
pip install -e ".[dev]"
```

## Usage

### Basic Test Run

```bash
# Run tests with visual progress (default)
pytest tests/

# Run specific test suite
pytest tests/e2e/cli/ -m e2e_cli

# Run with verbose output
pytest tests/ -v
```

### HTML Reports

HTML reports are automatically generated to `reports/test-report.html`:

```bash
# Reports are auto-generated, view with:
# Open reports/test-report.html in browser
```

### Custom Progress Plugin

The custom Rich progress plugin (`tests/pytest_rich_progress.py`) provides:
- Real-time progress bars
- Emoji indicators
- Color-coded output
- Summary tables

It's automatically enabled when:
- Running in a terminal (not CI)
- Not using `--quiet` flag
- Terminal supports colors

### Disable Visual Progress

To disable visual progress (useful for CI):

```bash
# Use quiet mode
pytest tests/ -q

# Or disable rich plugin
pytest tests/ -p no:rich_progress
```

## Configuration

### pytest.ini

The visual progress is configured in `pytest.ini`:

```ini
addopts =
    -v
    --color=yes
    -p tests.pytest_rich_progress  # Enable custom rich progress
    --html=reports/test-report.html
    --self-contained-html
```

### Environment Variables

- `NO_COLOR=1`: Disable colored output
- `TERM=dumb`: Disable rich formatting

## Library Versions

All libraries are pinned to latest stable production versions:

- **pytest**: 9.0.2 (Dec 2025)
- **rich**: 14.1.0 (Aug 2025)
- **pytest-html**: 4.1.1 (latest)
- **pytest-sugar**: 1.1.1 (latest)
- **pytest-rich**: 0.2.0 (Dec 2024)
