# QA Artifacts (BMAD Standard Path)

BMAD writes QA artifacts under `docs/qa/` (configured in `.bmad-core/core-config.yaml`).

## Folders

- `docs/qa/assessments/`: timestamped QA analyses (risk, test design, NFR, trace)
- `docs/qa/gates/`: gate decision YAMLs (PASS/CONCERNS/FAIL/WAIVED)
- `docs/qa/progressive/`: task-level progressive review YAMLs (optional)
- `docs/qa/background/`: background review logs (optional)

## Schemas / Templates

- Gate schema: `.bmad-core/templates/qa-gate-tmpl.yaml`
