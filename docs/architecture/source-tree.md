# Source Tree (Lean BMAD Shard)

This is a high-signal map of the repo so agents can navigate quickly.

## Top-Level

- **`tapps_agents/`**: main Python package (framework implementation)
- **`tests/`**: test suite (unit/integration/e2e)
- **`docs/`**: documentation
- **`workflows/`**: workflow YAMLs for the framework (not BMAD)
- **`templates/`**: framework templates (YAML/MD)
- **`scripts/`**: helper scripts (Python/PowerShell)
- **`examples/`**: example usage and sample configs
- **`.bmad-core/`**: BMAD methodology pack (agents/tasks/templates/config)

## Key Documentation Entrypoints

- **Project context**: `docs/PROJECT_CONTEXT.md`
- **Architecture**: `docs/ARCHITECTURE.md`
- **Developer guide**: `docs/DEVELOPER_GUIDE.md`
- **Cursor integration**: `docs/CURSOR_SKILLS_INSTALLATION_GUIDE.md`, `docs/CURSOR_RULES_SETUP.md`
