Code Review Workflow

Basic Code Review

Review a single file and get quality scores:

Review Command 
# Review a Python file
tapps-agents reviewer review src/api.py

# Output:
# {
#   "file": "src/api.py",
#   "scoring": {
#     "complexity_score": 6.5,
#     "security_score": 8.2,
#     "maintainability_score": 7.8,
#     "test_coverage_score": 7.0,
#     "performance_score": 7.5,
#     "overall_score": 74.2
#   },
#   "feedback": {
#     "summary": "Code quality is good with room for improvement...",
#     "issues": [...],
#     "recommendations": [...]
#   },
#   "passed": true
# }

Feature Implementation Workflow

Complete Feature Development

Use a workflow to implement a feature end-to-end:

Feature Workflow 
# Run full SDLC workflow
tapps-agents workflow run workflow-full-sdlc \
    --input description="Add user authentication endpoint"

# This workflow:
# 1. Plans the feature (Planner)
# 2. Designs the architecture (Architect)
# 3. Implements the code (Implementer)
# 4. Generates tests (Tester)
# 5. Reviews code quality (Reviewer)
# 6. Checks quality gates (Orchestrator)

Custom Workflow Example

Create a custom workflow for your specific needs:

Custom Workflow YAML 
name: API Endpoint Implementation
description: Implement a new API endpoint with tests

steps:
  - name: plan
    agent: planner
    command: plan
    args:
      description: "{{ input.description }}"
    
  - name: design
    agent: architect
    command: design
    depends_on: [plan]
    
  - name: implement
    agent: implementer
    command: implement
    args:
      description: "{{ plan.description }}"
    depends_on: [design]
    
  - name: test
    agent: tester
    command: generate
    args:
      file: "{{ implement.output_file }}"
    depends_on: [implement]
    
  - name: review
    agent: reviewer
    command: review
    args:
      file: "{{ implement.output_file }}"
    depends_on: [implement, test]
    
  - name: gate
    agent: orchestrator
    command: gate
    condition:
      review.overall_score >= 70

Multi-Agent Orchestration

Parallel Code Review

Run multiple reviewers in parallel for different aspects:

Multi-Agent Review 
# Python API example
from tapps_agents.core.multi_agent_orchestrator import MultiAgentOrchestrator

orchestrator = MultiAgentOrchestrator()

# Define parallel tasks
tasks = [
    {
        "agent": "reviewer",
        "command": "review",
        "args": {"file": "src/api.py", "focus": "security"}
    },
    {
        "agent": "reviewer",
        "command": "review",
        "args": {"file": "src/api.py", "focus": "performance"}
    },
    {
        "agent": "reviewer",
        "command": "review",
        "args": {"file": "src/api.py", "focus": "maintainability"}
    }
]

# Execute in parallel
results = await orchestrator.execute_parallel(tasks)

# Aggregate results
aggregated = orchestrator.aggregate_results(results)

Python API Examples

Using Agents Programmatically

Python API 
import asyncio
from tapps_agents.agents.reviewer.agent import ReviewerAgent

async def review_code():
    reviewer = ReviewerAgent()
    
    try:
        # Activate agent
        await reviewer.activate()
        
        # Review a file
        result = await reviewer.run("review", file="src/main.py")
        
        # Check scores
        if result.get("passed"):
            print(f"Code passed with score: {result['scoring']['overall_score']}")
        else:
            print(f"Code failed with score: {result['scoring']['overall_score']}")
            print(f"Issues: {result['feedback']['issues']}")
    
    finally:
        await reviewer.close()

# Run
asyncio.run(review_code())

Workflow Execution

Workflow API 
from tapps_agents.workflow.executor import WorkflowExecutor

async def run_workflow():
    executor = WorkflowExecutor()
    
    # Load workflow
    workflow = executor.load_workflow("workflows/feature-implementation.yaml")
    
    # Execute with inputs
    result = await executor.execute(
        workflow,
        inputs={"description": "Add payment processing"}
    )
    
    # Check results
    for step_name, step_result in result.step_results.items():
        print(f"{step_name}: {step_result.status}")
    
    return result

asyncio.run(run_workflow())

Python API

Enhancer API 
from tapps_agents.agents.enhancer.agent import EnhancerAgent

async def enhance_prompt():
    enhancer = EnhancerAgent()
    await enhancer.activate()
    
    # Full enhancement
    result = await enhancer.run(
        "enhance",
        prompt="Create a REST API for user management",
        output_format="markdown"
    )
    
    print(result["enhanced_prompt"])
    
    await enhancer.close()

asyncio.run(enhance_prompt())

from tapps_agents.core.agent_base import BaseAgent

class CustomAgent(BaseAgent):
    def __init__(self):
        super().__init__(
            agent_id="custom",
            agent_name="Custom Agent"
        )
    
    def get_commands(self):
        return [
            {"command": "*analyze", "description": "Analyze code"},
            {"command": "*transform", "description": "Transform code"}
        ]
    
    async def run(self, command: str, **kwargs):
        if command == "analyze":
            return await self._analyze(**kwargs)
        elif command == "transform":
            return await self._transform(**kwargs)
        else:
            return {"error": f"Unknown command: {command}"}
    
    async def _analyze(self, **kwargs):
        # Your analysis logic
        return {"status": "success", "analysis": "..."}
    
    async def _transform(self, **kwargs):
        # Your transformation logic
        return {"status": "success", "transformed": "..."}
