# TappsCodingAgents HTML Documentation

Modern, comprehensive HTML documentation for TappsCodingAgents framework.

## Files

- **index.html** - Main landing page with overview and navigation
- **user-guide.html** - Complete user guide with step-by-step instructions
- **technical-spec.html** - Comprehensive technical specification with algorithms and architecture
- **examples.html** - Real-world examples and use cases
- **api-reference.html** - Complete API documentation for CLI and Python
- **styles.css** - Modern 2025 styling with dark theme

## Viewing the Documentation

### Option 1: Open Directly
Simply open `index.html` in your web browser. All pages are linked together with navigation.

### Option 2: Local Server
For best experience, serve via a local web server:

```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000/index.html
```

### Option 3: VS Code Live Server

## Customization

The styling can be customized by editing `styles.css`. The design uses CSS variables for easy theming:

```css
:root {
    --primary: #6366f1;
    --bg-primary: #0f172a;
    --text-primary: #f1f5f9;
    /* ... more variables */
}
```
