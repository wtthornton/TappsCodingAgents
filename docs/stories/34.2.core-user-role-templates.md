## Story

**As a** framework user,
**I want** role templates for common developer roles,
**so that** I can quickly configure agents for my role (senior dev, junior dev, tech lead).

## Acceptance Criteria

1. `templates/user_roles/` directory is created
2. Senior developer template is created with appropriate configuration
3. Junior developer template is created with appropriate configuration
4. Tech lead template is created with appropriate configuration
5. Role-specific agent configurations are defined for each template
6. Templates follow defined format from Story 34.1
