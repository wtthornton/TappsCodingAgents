# Story 16.1: Custom Skill Template Generator

<!-- Source: implementation/EPIC_16_Custom_Skills_Support.md -->

## Status

**In Progress**

## Story

**As a** framework user,
**I want** a CLI command to generate custom Skill templates with all required fields,
**so that** I can easily create new Skills that integrate with the framework.

## Acceptance Criteria

1. Skill template generator CLI command is created
2. Template includes all required fields (name, description, allowed-tools, model_profile, Identity, Instructions, Commands, Capabilities, Configuration)
3. Template customization options are available (agent type, capabilities)
4. Template examples and documentation are created
5. Generator works correctly and produces valid Skill files

## Dev Notes

### Existing System Context

- Skills are stored in `.claude/skills/{skill_name}/SKILL.md`
- Skill format: YAML frontmatter + markdown content
- Built-in Skills in `tapps_agents/resources/claude/skills/`
- CLI commands in `tapps_agents/cli/`
- Template system exists for other artifacts

### Integration Approach

- Add command to `tapps_agents/cli/commands.py` or create new module
- Use existing template system patterns
- Follow built-in Skill format exactly
- Output to `.claude/skills/{custom_skill_name}/SKILL.md`
- Integrate with CLI argument parsing

### Technical Details

- **CLI Command**: `tapps-agents skill-template [--name=NAME] [--type=TYPE] [--interactive]`
- **Template Location**: `tapps_agents/resources/templates/skill_template.md`
- **Output Location**: `.claude/skills/{skill_name}/SKILL.md`
- **Required Fields**: name, description, allowed-tools, model_profile
- **Standard Sections**: Identity, Instructions, Commands, Capabilities, Configuration

### Risk Assessment

- **Primary risk**: Template may not match Cursor's expected format
  - **Mitigation**: Use exact format from built-in Skills, validate against examples
- **Primary risk**: Generated Skills may not work with framework
  - **Mitigation**: Follow built-in Skill patterns exactly, test generated Skills

### Rollback Plan

- Remove CLI command
- Remove template files
- No impact on existing Skills

### Testing

- Test template generation with various options
- Validate generated Skill format matches built-in Skills
- Test generated Skills load correctly
- Test CLI command error handling
