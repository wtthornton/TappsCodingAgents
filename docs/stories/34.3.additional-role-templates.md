## Story

**As a** framework user,
**I want** role templates for product manager and QA engineer,
**so that** non-developer team members can use agents effectively.

## Acceptance Criteria

1. Product manager template is created with PM-specific configurations
2. QA engineer template is created with QA-specific configurations
3. PM template emphasizes verbose Analyst/Planner, business-focused Architect
4. QA template emphasizes test-first approach, edge case focus
5. Templates follow defined format
